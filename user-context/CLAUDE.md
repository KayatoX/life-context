# User Context — ユーザーが管理するコンテキスト

ユーザー自身が手で書いて管理するコンテキスト置き場。
LLMに薄められていない、自分だけの一次情報を保管する。

## ⚠️ エージェントへの注意

**このディレクトリにエージェントが無断で書き込むことを禁止する。**
ここはユーザーが書く場所。`assistant/` とは明確に役割が異なる。

- ✅ 読むのはOK（コンテキストとして参照してよい）
- ❌ 書き込み・上書きはNG（ユーザーの指示がある場合のみ）

## ファイル一覧（例）

| ファイル | 内容 |
|---|---|
| `profile.md` | 基本情報・使っているツール・AIに知っておいてほしいこと |
| `mindset.md` | 価値観・思考スタイル・仕事の進め方 |

自分の生活・仕事・思考に合わせてファイルを追加していく。

## assistant/ との違い

| | `user-context/` | `assistant/` |
|---|---|---|
| 書く人 | ユーザー | エージェント |
| 内容 | 自分の言葉・経験・価値観 | 対話から観察したインサイト |
| 更新タイミング | ユーザーが好きなときに | コミット前にエージェントが判断 |
