# Project — プロジェクト管理ディレクトリ

特定スコープでエージェントを動かすためのディレクトリ。
各プロジェクトに CLAUDE.md を配置し、Claude Codeの遅延読み込みでアクセス時に自動的にコンテキストが渡される。

## 構造

```
project/
├── CLAUDE.md                  # 本ファイル
├── [project-name]/            # 進行中プロジェクト（各ディレクトリにCLAUDE.md必須）
│   ├── CLAUDE.md              #   プロジェクトの目的・フェーズ・主要ファイルを記載
│   └── ...
└── archive/                   # 完了済みプロジェクト
```

## 運用ルール

- プロジェクト作成時は必ず CLAUDE.md を配置する（10〜20行程度）
- 完了したプロジェクトは `archive/` に移動する
- README.md ではなく CLAUDE.md に統一する（Claude Codeが自動認識するため）

## プロジェクト CLAUDE.md のサンプル

```markdown
# [プロジェクト名]

[概要: 何のためのプロジェクトか1〜2行で]

## 現在のフェーズ

[例: 調査中 / 執筆中 / レビュー待ち / 完了]

## 主要ファイル

- `main.md` — [説明]
- `draft.md` — [説明]
```
