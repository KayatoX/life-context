# Agents — カスタムサブエージェント

Claude Codeが特定の役割を持つ専門エージェントとして動くための定義ファイル群。

## サブエージェントとは

通常のClaude Codeはひとつのセッションで会話しながら作業する。
サブエージェントは「専門家を別に呼び出す」イメージで、メインセッションとは独立して動く。

**メリット:**
- メインセッションのコンテキストウィンドウを消費しない
- 複数の作業を並行して走らせられる
- 役割を絞ることで精度が上がる

## 使い方

メインセッションで「researcherエージェントに○○を調べさせて」と指示するだけ。
Claude Codeがこのフォルダの定義を読んで、専門エージェントとして起動する。

## ファイルの作り方

`.claude/agents/[agent-name].md` を作成するだけ。
中身はエージェントへの指示書（役割・使えるツール・出力フォーマット等）を書く。

```markdown
# [エージェント名]

[役割の説明]

## できること

## 出力フォーマット
```
