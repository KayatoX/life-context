# Scrapbook — 汎用調べごと・インプット記録

特定プロジェクトに紐づかないリサーチ、ニュース分析、調べごとを格納する場所。

## ディレクトリ構造

```
scrapbook/
├── daily/                    # テーマ振り分けなし（日別）
│   └── YYYY-MM-DD/
│       └── [トピック名].md
└── theme/                    # テーマ別に体系的に蓄積
    └── [theme-name]/
        └── [トピック名].md
```

## 振り分けルール

- **同じトピックで複数回調査する or 今後もする** → `theme/[theme-name]/` に格納
- **単発のニュース分析・一回きりの調べごと** → `daily/YYYY-MM-DD/` に格納

## ファイル命名規則

- kebab-case推奨（英数字・ハイフン・アンダースコアのみ）
- 内容がわかるトピック名をつける（例: `claude-skills-overview.md`）
