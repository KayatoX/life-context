# Assistant — エージェント自律管理領域

エージェント（Claude Code）が自律的に情報を蓄積・更新するためのディレクトリ。
ユーザーが直接編集するのではなく、エージェントが主体的に管理する。

## ディレクトリ構造

```
assistant/
├── CLAUDE.md                        # 本ファイル
├── user-insights.md                 # ユーザーインサイト（思考・価値観・行動傾向）
└── log/                             # エージェント操作ログ
    └── YYYY-MM-DD/                  #   日別ログ・意思決定記録
```

## 自律更新対象

### user-insights.md
- ユーザーの思考パターン、価値観、コミュニケーション傾向、行動特性の統合分析
- **更新タイミング**: Git操作時（コミット前）にセッション中の新たな観察を反映するか判断
- `user-context/` の内容はあくまで参考。対話から得たインサイトのみここに記録する

### log/
- 意思決定ログ（gitのdiffだけでは伝わらない「なぜそうしたか」を残す）
- 日付別ディレクトリに `decisions.md` として記録
